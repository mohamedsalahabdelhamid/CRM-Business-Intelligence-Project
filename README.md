📊 CRM Analytics Dashboard

📝 Overview
This project focuses on building an interactive CRM Dashboard using Power BI to provide actionable insights into customer relationship management. The dashboard helps stakeholders monitor sales performance, customer engagement, and team productivity, turning raw CRM data into clear visualizations that support data-driven decision making.

🎯 Objectives
- Clean and transform CRM data to improve quality and consistency.
- Build a robust data model using Power Query and DAX.
- Create custom DAX measures for advanced analytics.
- Design a user friendly and interactive dashboard.
- Apply best practices in data visualization to highlight key metrics.

📊 Key Insights
- Customer Segmentation: Identify top customer groups contributing to revenue.
- Sales Performance: Track total sales, pipeline progress, and conversion rates.
- Team Productivity: Compare performance across managers and sales agents.
- Regional Analysis: Visualize sales distribution across different regions.
- Customer Retention: Monitor repeat customers and churn risk.

🛠 Tools & Technologies
- Power BI: For dashboard creation and visualization.
- Power Query: For data cleaning and transformation.
- DAX: For building custom measures and calculations.
- Data Modeling: For structuring CRM data effectively.

📂 Project Structure
- Data : Raw CRM dataset.
- Reports : Power BI report files (.pbix).
- Images : Screenshots of the dashboard.
- README.md : Project documentation.

🚀 How to Use
1. Open the pbix file in **Power BI Desktop**.
2. Connect or refresh the CRM dataset.
3. Explore the dashboard using filters and interactive visuals.

📈 Value
This project demonstrates skills in:
- Data cleaning and transformation.
- Data modeling and DAX calculations.
- Dashboard design and visualization best practices.
- Delivering insights that improve customer management and sales strategies.
